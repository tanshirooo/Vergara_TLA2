Flexible Grid

Media Queries

CSS media queries handle layout changes for different breakpoints.

Media Query Example

Here's how to write responsive CSS:

/* Mobile First - Base styles */
.container {
  width: 100%;
  padding: 0 1rem;
}

/* Tablet (768px and up) */
@media (min-width: 768px) {
  .container {
    width: 750px;
  }
}

/* Desktop (992px and up) */
@media (min-width: 992px) {
  .container {
    width: 970px;
  }
}
